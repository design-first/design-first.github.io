  • Descriere: Pune un ID unic unei componente.

  • Descriere: Elimina un ascultător.
  • Schema: _Component

  • Descriere: Adaugă un eveniment.
  • Schema: _Component

  • Description: Pune un component cu ID-ul unic.
  • Schema: _Component
