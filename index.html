WEB IDE

DESIGN ON THE GO

You can begin your design on your Mac and then continue it on your iPad.

GITHUB

System Designer has a GitHub module to synchronize your work between all these apps.

Run
Your App

Save and run

Once you have designed your application, run it with System Runtime, the runtime of your application.

TYPE CHECKING

System Runtime checks at runtime if the signatures of invoked methods are compliant with your design.

MANAGE YOUR APPLICATION

System Runtime acts as an ODM (Object-Document Mapper) to manage your components as NoSQL Documents.

UNIVERSAL JAVASCRIPT

Your application can be run server-side or client-side with no modification.

Compose the models and the components of your application.
